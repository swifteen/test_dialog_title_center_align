# test_dialog_title_center_align
参考https://www.qtcentre.org/threads/25974-stylesheet-to-QDialog-Title-Bar

- 实现自定义的窗口标题栏，将标题居中显示
- 父类标题栏自带关闭按钮，子窗口不必包含关闭按钮代码，只需要重写closeEvent
- 父类实现可拖动的窗口
- qss实现圆角矩形风格
